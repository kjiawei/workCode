二室尿素清除期指数

( Kt/V2-Pool)

 

 

 

 

( Kt/V) > 1.2提示透析充分。

 

(仅供参考!请结合临床具体情况分析)

 

Dt/V＝∑ Q∣Cdi-Cdo∣/∣Cdimax-Cdomin∣

Q =[Do/Di*Bo/Bi*(0.58W+UF)/(0.58W-UF)]*(1.73/Po）

Cdi及Cdo为透析液入口及出口电导(mS/cm);

Do: 实际透析液流量; Di标准透析液流量(500ml/min);Bo: 实际血流量; Bi: 标准血流量(255ml/min);Po: 实际体表面积; UF:超滤量; UF:干体重;

透析尿素下降率

 (URR)

 

> 70(%)

URR只反映溶质下降百分率，不是实际溶质清除量, > 70(%)提示透析充分。

(仅供参考!请结合临床具体情况分析)

URR= 1 -  (Ct / Co)　＝　1 - R

* Ct: 第一次透析后BUN(mmol/L)

* Co: 第一次透析前BUN(mmol/L)

一室尿素清除指数 

( Kt/V1-Pool)

 

> 1.2

指一次透析中清除尿素的容积占总体水的比例。（Kt/V1-Pool ）> 1.2提示透析充分。

(仅供参考!请结合临床具体情况分析)

Kt/V1-Pool =-In [R –(0.08 x t) ]+ [4-(3.5 x R)] x UF/W

* R= Ct / Co;　 t:透析时间；

UF：超滤量(L);　W:干体重(Kg)

二室尿素清除期指数

 ( Kt/V2-Pool)（Kt/Vdp）

 

> 1.0

同时考虑透析过程中尿素产生、体液容量变化以及溶质室间转运系数等变量因素。( Kt/V2-Pool) > 1.0提示透析充分。

(仅供参考!请结合临床具体情况分析)

Kt/V2-Pool  = ( Kt/V1-Pool) x [1 – (36/ t) ] + 0.03

 

t: 透析时间(min)

二室溶质清除指数 

(SRI2Pool)

 

> 70%

SRI是通过二室尿素动力学模型求出的参数,其表示透析中溶质排除量. SRI> 70%,提示透析充分。

(仅供参考!请结合临床具体情况分析)

(SRI2Pool) = [1 – Ct  x  Vtd / ( Co x Vo )] x 100%

Ct: 第一次透析后BUN(mmol/L); Co: 第一次透析前BUN(mmol/L); Vtd: 透析后容积; Vo:透析前容积;

Vtd = (R-Co x UF) / (Co-Ct);  Vo = Vtd + UF

标准蛋白质分解率 

（nPCR）

 

 

 

 

> 1.1 (g/kg .d)

l 透析患者蛋白质摄入量宜> 1.1 (g/kg .d)，这样，nPCR也应> 1.1 (g/kg .d)。

l nPCR <0.8 (g/kg .d) 提示营养不良，透析不充分概率高，请结合Kt/V值进行临床分析。

(仅供参考!请结合临床具体情况分析)

nPCR = ( 9.35 G + 0.29 W ) / W ¸ 0.58

G =（Co2 - Ct）x W /q + V x Cu / q 

W:干体重(Kg)；V:透析间期尿量(ml);  q:透析间隔时间(min); Cu: q间期内尿中BUN; Ct:第一次透析后BUN(mmol/L)；Co2: 第二次透析前BUN(mmol/L)

时间平均尿素浓度

（TACurea）

 

 

< 50（mg/dl）

反映了透析尿素清除量与患者蛋白质代谢的综合情况，为透析充分性的参考指标。TACurea>50（mg/dl）提示心血管、消化道、神经系统等远期并发症增加。(仅供参考!请结合临床具体情况分析)

TACurea =[（Co + Ct）Td+（Ct+ Co2）Id] /2（Td+ Id）

Co: 第一次透析前BUN(mmol/L)；Ct:第一次透析后BUN(mmol/L)；Co2: 第二次透析前BUN(mmol/L)；

Td：透析时间（h）； Id：透析间隔时间（h）

病人残余肾功能

(Kru)

 

 

 

 

 

(ml/min)

 

Kru(mmol/L)=( V/q ) x [ 2Cu / (Ct+Co2) ]

V:透析间期尿量(ml);  q:透析间隔时间(min);

Cu: q间期内尿中BUN; Ct:第一次透析后BUN(mmol/L)；Co2: 第二次透析前BUN(mmol/L)

一室尿素清除指数与病人残余肾功能之和

(DT)

 

 

(ml/min)

 

DT(HD3次/周) =Kt/V1-Pool(或机测Kt/V)+(5.5xKru/V)

DT(HD2次/周) =Kt/V1-Pool(或机测Kt/V)+(9.5xKru/V)

V:透析间期尿量(ml)

 

Dt/V＝∑ Q∣Cdi-Cdo∣/∣Cdimax-Cdomin∣

Q = [ Do/Di * Bo/Bi * (0.58 W + UF)/(0.58 W - UF)] ¸ P

Do: 实际透析液流量; Di标准透析液流量(ml/min);

Bo: 实际血流量; Bi: 标准血流量(ml/min);

P: 体表面积; UF:超滤量; UF:干体重;
